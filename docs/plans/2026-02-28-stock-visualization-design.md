# 股票每日复盘可视化看板 · 设计文档

**日期**：2026-02-28
**状态**：已确认，待实施

---

## 需求概述

基于舞阳复盘网站（wuylh.com）的涨停数据，构建一个本地运行的每日股票复盘可视化看板。
看板每次启动时自动检测并拉取最新数据，通过图表直观展示当日市场核心信息。

---

## 技术栈

| 层次 | 技术 |
|------|------|
| 看板框架 | Streamlit |
| 图表库 | Plotly |
| 数据处理 | pandas |
| 爬虫 | requests + BeautifulSoup（若页面为 JS 渲染则改用 playwright） |
| 数据存储 | 本地 JSON 文件（按日期命名） |

---

## 项目结构

```
daily-review/
├── app.py              # Streamlit 主入口，看板页面

├── start.bat           # Windows 双击启动脚本
├── requirements.txt    # 依赖列表
└── data/
    ├── 2026-02-28.json # 每日数据（按日期命名）
    └── ...
```

## 数据爬取

**数据来源**：`https://www.wuylh.com/replayrobot/index.html`

**爬取字段**：
- 涨停股列表：股票名、代码、连板数、涨停原因、最新价、涨幅、成交额、换手率
- 开板股列表：同上字段

**每日 JSON 结构**：

```json
{
  "date": "2026-02-28",
  "limit_up": [
    {
      "name": "某股票",
      "code": "000001",
      "continuous_days": 3,
      "reason": "人工智能",
      "price": 15.6,
      "change_pct": 10.01,
      "amount": 5.2,
      "turnover_rate": 12.3
    }
  ],
  "limit_broken": [],
  "fetched_at": "2026-02-28 16:00:00"
}
```

**扩展预留**：fetcher 模块设计为可插拔接口，后续可对接 Tushare / 聚宽等第三方 API，无需修改看板代码。

## 启动逻辑

```
启动 app.py
    ↓
检查 data/今日日期.json 是否存在？
    ├── 存在 → 直接加载，右上角显示"数据时间：HH:MM"
    └── 不存在 → 调用 fetcher.py 爬取
            ↓
        爬取结果是否为空（休市/节假日）？
            ├── 非空 → 保存 JSON，加载数据
            └── 为空 → 查找最近有数据的交易日 JSON
                        顶部显示黄色提示："今日休市，显示 YYYY-MM-DD 数据"
```

**错误处理**：
- 爬取失败（网络异常等）→ 页面显示红色警告，不崩溃
- 历史数据不足10天 → 趋势图显示现有天数

## 看板模块设计

### 顶部概览卡片

一排数字卡片，快速掌握全局：

| 涨停家数 | 开板家数 | 炸板率 | 最高连板 | 最强题材 |
|---------|---------|-------|---------|---------|
| 38 | 12 | 31.6% | 6板 | 人工智能 |

---

### 模块 A · 最高连板股（表格）

- 按连板数从高到低排列
- 最高连板行高亮标红
- 字段：股票名、连板数、涨停原因、成交额

---

### 模块 B · 题材热度排行（横向条形图）

- X 轴：涨停家数
- Y 轴：题材名称（从多到少排列）
- 鼠标悬停显示该题材所有涨停股名

---

### 模块 C · 近10日情绪趋势（折线图）

- 蓝线：涨停家数
- 橙线：开板家数
- 红线：炸板率 %（右轴）
- X 轴：日期，悬停显示具体数值

---

### 模块 D · 连板梯队图（柱状图）

- X 轴：连板数（1板、2板、3板……）
- Y 轴：股票数量
- 柱子上标注代表票名（如"4板：某票"）

## 启动方式

**首次安装**：
```bash
pip install -r requirements.txt
```

**日常使用**：双击 `start.bat`，浏览器自动打开 `http://localhost:8501`

`start.bat` 内容：
```bat
@echo off
streamlit run app.py
pause
```
