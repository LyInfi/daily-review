# 近10日情绪趋势图全量指标设计文档

日期：2026-02-28

## 需求

将"近10日情绪趋势"图扩展为显示 wuylh.com API `tenDays` 数组中全部30个指标，并让用户通过 `st.multiselect` 自由选择展示哪些指标。分两块展示，一目了然。

## 方案：双块 multiselect + 双轴折线图

### 块一：日级别指标图

- multiselect 列出索引 [1]-[8],[14],[19]-[30] 共21个指标
- 预选默认：总涨停、开板数、封板率%、连板率%
- 双轴折线图：左轴=家数/金额类，右轴=比率%类（静态映射）

### 块二：涨停时间分布图

- multiselect 列出索引 [9]-[13],[15]-[18] 共9个指标
- 预选默认：首板、二连板、三连板
- 单轴折线图（量纲统一，均为家数）

## 改动范围

| 文件 | 改动 |
|------|------|
| `fetcher.py` | 扩展 `_parse_ten_days()` 解析全部30个字段 |
| `processor.py` | 扩展 `_history_entry_from_10d()` 透传所有字段 |
| `app.py` | 重写模块C，新增时间分布图块 |
| `tests/test_fetcher.py` | 补充新字段断言 |
| `tests/test_processor.py` | 补充新字段断言 |

## 错误处理

- 老缓存文件缺少新字段时，默认填 0
- multiselect 选空时显示提示，不报错
- len=20 字段取前10个值，不足补0
